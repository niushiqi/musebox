## 🚀 快速开始

### 1. 安装插件

1. 下载 `批量重命名注释打标 - MuseAI.eagleplugin`
2. 双击安装,或在 Eagle 中选择"插件 > 安装插件"

### 2. 配置 API

1. 打开插件,切换到"设置"标签页
2. 选择 AI 服务商
3. 输入 API Key
4. 选择模型
5. 点击"测试连接"
6. 点击"保存设置"

### 3. 处理图片

1. 在 Eagle 中选择图片
2. 打开插件,切换到"工作台"
3. 启用需要的功能
4. 点击"开始处理"

详细说明请查看 [快速开始指南](快速开始.md)

## 💡 使用技巧

### 1. 选择合适的服务商
- **Google AI Studio**: 免费额度大,适合个人用户
- **火山方舟**: 国内访问快,适合国内用户
- **阿里云**: 企业级稳定性,适合商业用途

### 2. 优化处理速度
- 只启用需要的功能
- 选择较小的模型
- 减少最大 Token 数
- 分批处理图片

### 3. 控制成本
- 开启"跳过已有注释"
- 开启"跳过已处理"
- 定期查看使用统计
- 合理设置 Token 上限

### 4. 提高准确性
- 选择更强大的模型
- 增加最大 Token 数
- 使用自定义提示词

## 🐛 常见问题

### Q: API Key 在哪里获取?
A: 请访问对应服务商的官网注册获取,详见 [快速开始指南](快速开始.md)。

### Q: 为什么测试连接失败?
A: 请检查 API Key 是否正确、网络连接是否正常、服务商是否有余额。

### Q: 处理速度慢怎么办?
A: 这是正常现象,AI 处理需要时间。建议分批处理,每次不超过 20 张。

### Q: Token 会花钱吗?
A: 大部分服务商提供免费额度,Google AI Studio 免费额度很大。

### Q: 可以自定义提示词吗?
A: 当前版本使用默认提示词,自定义功能将在后续版本提供。

更多问题请查看 [快速开始指南](快速开始.md)
